# Starter Rubric：小红书种草 / 好物分享 — v0 cold-start 占位

适用于在小红书做**好物种草 / 测评 / 盘点 / 教程**的博主——你分享的**东西本身(银饰 / 香水 / 美妆 / 穿搭…)是产品**,图文或视频是载体。本文件以**银饰 + 香水**为默认垂类写锚点,其他品类同构,改锚点即可。

> **重要:这是给小红书种草的 v0 占位 rubric,不是抖音观点视频那套。**
> 内置的 [opinion-video.md](opinion-video.md) 是从一个**抖音观点视频博主、按播放量**拟合的——维度(情感/社会议题/讽刺)和分档(万播放)都不适合种草。**那份保留不动**,种草请用本文件。
> 完全没数据的新号:本文件会打错,前 5 篇精度大概 ±50%,这是 cold-start 的数学事实,不是 rubric 失败。

---

## 北极星:小红书不看播放,看**收藏**

抖音/YouTube 的命脉是播放与完播;**小红书种草的命脉是收藏(藏曝比)**。原因:

- **收藏 = 强购买/参考意图**。用户"存起来"是为了之后买、对比、照着做——这是种草最值钱的信号。
- **算法把收藏当强正反馈**,高藏曝比的笔记更容易被持续推 + 进搜索池。
- **小红书是搜索引擎**:一篇好种草笔记在发布 3 天后仍会因为"s925 不发黑""木质调香水 平替"这类搜索词被反复翻出来,长尾远比抖音长。

所以本 rubric 的维度都是为"**让人想点进来 → 想收藏 → 评论问链接**"这条链路设计的,**不是**为"让人看完转发"。

---

## v0 综合分公式(**等权占位**)

```
composite = (CA + SV + SK + TR + VS + PS + IH) / 7 × 2.0
```

每个维度 0-5 整数分,综合分 0-10。

**为什么等权**:你还没有自己账号的数据支持任何特定权重。等权让你对每个维度都给"中立关注",跑完 5 篇复盘后,你的藏曝比数据会告诉你哪几个维度真的预测了你的收藏——那时再 `/cheat-bump` 拟合你自己的权重(种草号通常会发现 **CA 封面 与 SV 收藏价值** 权重最高)。

## 7 个维度(图文 + 视频通用)

### CA — Cover Attraction(封面吸引力)
*在信息流里,封面第一眼能不能让人**停下来点进去**?*

小红书是**封面驱动**的平台——95% 的流量决定发生在用户划到你封面的 0.5 秒内。这是种草号最强的流量杠杆。

- **0** — 随手拍、杂乱背景、看不出主体;或纯文字大字报无记忆点
- **3** — 主体清晰、构图干净,但"看过就忘",和别人家长一样
- **5** — 一眼抓人 + 有信息钩子。银饰:出片质感/反光细节/上身对比/"这条我天天戴";香水:瓶身美学/色调氛围/"夏天用它被问了 8 次"。封面常带一句**利益点文案**(平替/显白/不发黑/留香一整天)

5/5 锚点:封面同时做到"好看"+"给一个点进来的理由"。0 分对照:好看但没理由(纯产品图,像电商主图)。

### SV — Save Value(收藏价值,**北极星维度**)
*这篇值不值得用户"**存起来以后用**"?*

收藏价值高的内容有共同结构:**可复用、可对照、能省事**。

- **0** — 单纯晒一件东西,看完即走,没有"留着有用"的理由
- **3** — 有一点干货(一条戴法 / 一个香调描述),但不成体系
- **5** — 天然高收藏的形态:**测评对比 / 盘点合集 / 避雷清单 / 平价替代 / 香调地图 / 叠戴公式 / 不同场合选香 / 尺寸保养教程**。用户心想"先收藏,周末逛街/下单时再看"

5/5 锚点:「10 支夏季淡香分价位盘点」「s925 银饰防氧化保养 5 步」「初学者叠戴公式·手腕篇」。0 分对照:「今天买了个戒指好开心」。

### SK — Search Keywords(标题/关键词搜索性)
*标题和正文有没有埋用户**真的会去搜**的词?*

小红书一半以上流量来自搜索。不埋词 = 放弃长尾。

- **0** — 标题是情绪自言自语(「啊啊啊好喜欢」),无任何可被搜索的实词
- **3** — 有品类词(「银戒指」「香水」),但太泛、竞争大
- **5** — 精准长尾词组合:**品类 + 属性 + 人群/场景**。银饰:「s925」「不发黑」「显白」「小众设计」「养耳洞」「敏感肌耳钉」;香水:「养肤香」「木质调」「办公室淡香」「约会香」「平替」「留香持久」「闻香报告」

> 速记:用户**焦虑或欲望**的时候会在搜索框打什么字,那就是你的关键词。

### TR — Trust / Authenticity(真实信任感)
*看起来像**真人真实使用**,还是像广告?*

种草最怕假。信任感直接决定评论区是不是有人问"链接""求闻香报告"。

- **0** — 通篇彩虹屁、无缺点、像品牌通稿
- **3** — 有真实使用描述,但优缺点不平衡
- **5** — 有具体使用痕迹 + **敢说缺点/适用边界**(「这支留香只有 4 小时,但便宜到无所谓」「这条链子细,运动时摘」「黄皮慎入这个色」)。有对比、有真实场景照,不完美反而可信

### VS — Visual Style(出片 / 视觉氛围)
*图片/视频有没有**统一调性和质感**?*

决定点赞、二次曝光,以及账号辨识度(种草号靠视觉建立"审美信任")。

- **0** — 曝光/色调忽明忽暗,每张风格不一,像不同人拍的
- **3** — 干净但平淡,没有个人风格
- **5** — 统一色调/质感 + 细节到位。银饰:微距反光、皮肤质感、ins 风/复古风一致;香水:氛围摆拍、光影、季节感。**滑到第二张第三张还想看**

### PS — Pain / Scene(痛点 / 场景代入)
*有没有戳中一个**具体的场景或痛点**,让人觉得"这说的就是我"?*

- **0** — 泛泛而谈,没有具体人群或场景
- **3** — 提到一个场景但不够具体
- **5** — 锐利具体的场景/痛点:「夏天香水两小时就没了怎么办」「银饰戴几天就发黑」「敏感肌耳洞反复发炎」「不知道约会该喷什么」「预算 200 想要高级感」。痛点越具体,评论区越炸

### IH — Interaction Hook(互动钩子)
*有没有引导用户**评论/提问/站队**?*

小红书把评论互动当强权重,评论区热度还会反哺曝光。

- **0** — 没有任何引导,结尾就没了
- **3** — 有一句软引导(「你们觉得呢」)
- **5** — 具体可回答的钩子:「你们想要平替版还是大牌版,点赞过 X 出下集」「报你的肤质我推对应的香」「评论区交作业晒你的叠戴」。**让评论本身有内容可写**

## 视频笔记附加维度(**仅当这篇是视频时加**)

你说想试着做视频——小红书视频笔记叠加下面 2 个维度,图文笔记**跳过、不计入公式**。

### VH — Video Hook 3s(前 3 秒钩子)
*视频开头 3 秒能不能留住人?*
- **0** — 慢热铺垫、片头 logo、「大家好今天给大家分享」
- **3** — 直接进主体
- **5** — 第一帧就是结果/对比/冲突(「这三支香被问最多」直接怼脸闻香表情 / 银饰前后对比),配封面级首帧

### VC — Completion / Pace(完播 / 节奏)
*时长和信息密度匹配吗,会不会拖?*
- **0** — 又长又水,大量废镜头
- **3** — 节奏一般
- **5** — 紧凑,15-30s 讲清一个点,每 2-3 秒一个信息/画面变化

> 视频版公式:`composite = (CA + SV + SK + TR + VS + PS + IH + VH + VC) / 9 × 2.0`。
> 图文版用上面的 /7 版本。两种笔记在复盘时**分开记锚点**——它们的爆款逻辑不同。

## Bucket 预测:小红书口径(比率桶,适用所有阶段)

> ⚠️ 小红书 0 粉新号笔记曝光典型几百到几千,**不能套抖音"5w 是底部"那种绝对桶**,否则每篇都落底部 99%。用相对自己上一篇的**比率桶**。
> bucket 仍以**曝光(view_count)**分档(和工具其余部分对齐、可比);**收藏率(藏曝比)单独作质量信号**看(见下一节),不混进 bucket。

### 第 1 篇:用小红书新号通用默认

| Bucket | 范围(实际曝光数)| 含义 | 先验概率 |
|---|---|---|---|
| 底部 | < 300 | 基本没进流量池 | 30% |
| 基础盘 | 300 - 2,000 | 小推荐,封面/标题及格 | 40% |
| 命中 | 2,000 - 10,000 | 进了二级流量池,有破圈苗头 | 20% |
| 小爆 | 10,000 - 100,000 | 收藏率撑住的破圈 | 8% |
| 大爆 | > 100,000 | 进大流量池 / 搜索长尾叠加 | 2% |

### 第 2 篇起:比率桶(baseline = 上一篇曝光,或最近 3 篇中位数)

| Bucket | 倍数范围 | 含义 |
|---|---|---|
| 退步 | < 0.3 × baseline | 比上一篇明显差 |
| 持平 | 0.3 - 1 × baseline | 同档 |
| 命中 | 1 - 3 × baseline | 中度突破 |
| 小爆 | 3 - 10 × baseline | 显著破圈 |
| 大爆 | > 10 × baseline | 量级跃迁 |

> calibration_samples 少时,概率分布**应该更平**(如 30/30/20/15/5),诚实表达不确定——不靠省略 bucket 解决。

## 收藏率质量带(**小红书专属,看这个比看曝光重要**)

每次复盘除了 bucket,**必算三个比率**(xhs-explore 适配器直接给):

| 指标 | 算法 | 种草内容的健康参考带(先验,需用你自己数据重标) |
|---|---|---|
| **藏曝比(收藏率)** | 收藏 / 曝光 | < 1% 偏弱 · 1-3% 及格 · 3-8% 好 · >8% 很强 |
| **赞曝比(点赞率)** | 点赞 / 曝光 | < 2% 偏弱 · 2-5% 及格 · 5-10% 好 · >10% 很强 |
| **赞藏比** | 点赞 / 收藏 | 种草内容**收藏占比越高越好**(说明有购买/参考意图)。赞藏比 < 2 是优质种草信号;>5 偏"好看但不想要" |

> ⚠️ 上面的带是先验占位,**不是你账号的真值**。跑完 5 篇用你自己的中位数重设(`/cheat-bump --bucket-only` 之外,手动在 rubric_notes.md 记你账号的健康带)。
> **核心判断**:一篇曝光一般但藏曝比很高的笔记,比一篇曝光高但没人收藏的笔记**更值得复制**——前者击中了真实需求。

## T+N 复盘窗口:小红书要看两次

`RETRO_WINDOW_DAYS = 3`(默认,抓算法推荐期的结论)

**但小红书建议额外在 T+7d 再看一次**:小红书搜索长尾强,一篇笔记的收藏会在发布一周内**持续缓慢上涨**(被搜索翻出来)。只看 T+3d 会低估种草笔记的真实收藏价值。

- T+3d:判断"算法推荐期"表现(曝光、即时点赞)
- T+7d:判断"搜索长尾 + 收藏沉淀"(藏曝比通常此时才稳)

覆盖默认时记得写进 `rubric_notes.md`。

## Anchors(锚点样本)

打分新笔记时,找 2-4 个 composite 接近的旧笔记,列出它们的实际藏曝比/赞曝比对照。**锚点对比能抓出公式抓不到的错误**(比如"这两篇我都打 CA=5,但实际收藏差 5 倍——封面差异在哪")。

锚点是你边发边攒的。≥5 篇校准前锚点不可用,这是正常的。Cold-start 期预测文件里标 `锚点对比:N/A(cold-start)`。

## Cold-start 战略(**必读**)

前 5 篇的"预测"不是预测,是**数据采集**。理解这点是工具对你是否有用的分水岭。

**取样策略:主动选差异最大的笔记。** 别前 5 篇都发"同款合集",否则复盘时分不清是哪个维度在起作用。刻意各发一篇侧重不同维度的:
- 1 篇 **SV 主导**(硬核合集/测评)
- 1 篇 **CA 主导**(封面/出片极致,内容轻)
- 1 篇 **PS 主导**(强痛点场景,如"银饰发黑急救")
- 1 篇 **SK 主导**(主攻搜索词,如"XX 香水平替")
- 1 篇综合中等

复盘后你会看到 pattern(比如"我的封面分总是高估""我的合集藏曝比远超晒物")——这就是第一次 `/cheat-bump` 的证据。

**你不该做的**:cold-start 期看着 composite 8.4 就觉得这条会爆。精度 ±50%,决策没意义。已决定要发的发,犹豫的靠自己感觉。

| 校准样本 | 你能相信什么 |
|---|---|
| N=0-2 | 啥都别信,bucket 是占位 |
| N=3-5 | 信"哪个维度可能重要"的方向 |
| N=5-10 | 信 bucket 排序;不信中枢点估计 |
| N=10-20 | 中枢可信 ±30%,可作决策参考之一 |
| N≥20 | rubric 真正成为"作弊器" |

## 第 6 篇起的升级选项

跑完 5 篇校准后:
- **路径 A**:`/cheat-bump --propose "<权重调整>"` 自己拟合(种草号常把 CA/SV 提权)
- **路径 B**:把藏曝比作为主回归目标重设权重
- **路径 C**:5 篇还看不出 pattern → 再跑 5 篇,等权 v0 在 10-15 样本前不丢人

## 这份 rubric 注定会打错的地方(已知 limitations)

1. **品类敏感**:银饰(看出片/设计)和香水(看气味描述/场景,图片只能传氛围)的爆款逻辑其实不同——跑够样本后可能要给银饰、香水**各拆一份**权重
2. **没有"带货转化"维度**:本 rubric 预测的是**收藏/互动**,不是 GMV。高收藏 ≠ 高转化,转化还受选品/价格/挂车影响,需另行追踪
3. **季节性强**:香水分春夏秋冬,银饰分日常/节日礼赠——同一选题不同季节藏曝比差很多,rubric 没捕捉时令
4. **封面玄学**:CA 维度只能粗评,真实封面 A/B 效果只能发出去才知道——封面单独多准备 2-3 版测

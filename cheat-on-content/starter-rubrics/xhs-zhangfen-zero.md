# Starter Rubric：小红书涨粉 / 口碑 — v0 cold-start 占位

适用于现阶段目标是**涨粉 + 攒口碑**(不是带货转化)的小红书博主——你做的是**有价值的真实分享**,银饰 / 香水 等只是**人设载体**,不是要安利的产品。本文件以**银饰 + 香水审美/生活方式**为默认垂类写锚点,其他品类同构,改锚点即可。

> **重要:这是给"涨粉/口碑"阶段的 v0 占位 rubric,北极星是「关注转化率」,不是收藏率。**
> 同目录的 [xhs-zhongcao-zero.md](xhs-zhongcao-zero.md) 是**种草号**那套(北极星=收藏率/藏曝比),维度为"让人想收藏→评论问链接"设计——**那份保留不动**,等你哪天要带货再切回。本阶段用本文件。
> 内置的 [opinion-video.md](opinion-video.md) 是抖音观点视频按播放量拟合的,与这里无关。
> 完全没数据的新号:本文件会打错,前 5 篇精度大概 ±50%,这是 cold-start 的数学事实,不是 rubric 失败。

---

## 北极星:这个阶段不看收藏,看**关注转化 + 口碑**

种草号的命脉是收藏(藏曝比);**涨粉阶段的命脉是「有多少人因为这篇关注了你」**。这两个指标会打架——一篇高收藏的盘点,大家把**产品**存走了,**不会关注你这个人**。所以本 rubric 换一套信号:

- **关注转化率(北极星)= 新增关注 / 阅读(或曝光)**。它直接回答"这篇有没有把路人变成粉丝"。人为什么点关注?因为他觉得**你这个账号/人设以后还会持续给我价值**——靠的是强人设、连载感、态度共鸣,不是单篇有用。
- **分享率(口碑副信号)= 分享 / 曝光**。愿意转发给朋友 = 真心觉得你好、愿意替你安利,**比收藏更代表口碑**(收藏是"留着自己用",分享是"想让别人也认识你")。
- **评论正向率(口碑副信号)= 正向评论 / 总评论**。评论区是真情绪还是冷场/杠,直接反映口碑温度。
- **收藏降级**:收藏仍看,但只当"这篇有用值得回看"的**副信号**,**不再是目标**。别为了收藏去做盘点/平替安利,那会把你带回种草号。

所以本 rubric 的维度都是为"**让人点进来 → 记住你这个人 → 关注 + 愿意安利你**"这条链路设计的,**不是**为"让人收藏产品"。

## v0 综合分公式(**等权占位**)

```
composite = (CA + ID + FR + EM + TR + SH + IH) / 7 × 2.0
```

每个维度 0-5 整数分,综合分 0-10。

**为什么等权**:你还没有自己账号的数据支持任何特定权重。等权让你对每个维度都给"中立关注",跑完 5 篇复盘后,你的**关注转化率**数据会告诉你哪几个维度真的预测了涨粉——那时再 `/cheat-bump` 拟合你自己的权重(涨粉号通常会发现 **ID 人设 与 FR 关注理由** 权重最高,而非封面)。

## 7 个维度(图文 + 视频通用)

### CA — Cover Attraction(封面 / 首图吸引力)
*在信息流里,封面第一眼能不能让人**停下来点进去认识你**?*

小红书 95% 的流量决定发生在用户划到封面的 0.5 秒内。但涨粉阶段的封面不是"被安利的产品图",而是"**想点进来看看这个人**"。

- **0** — 杂乱随手拍 / 纯产品电商图 / 大字报无记忆点
- **3** — 干净好看,但和别人家长一样,看不出"是谁"
- **5** — 一眼抓人 + 带**人设/态度记忆点**。银饰:有你独特审美的上身氛围、一句态度文案("我只戴银,十年了");香水:有调性的氛围 + 个人视角。封面让人觉得"这个账号有点东西,点进去看看"

5/5 锚点:封面同时"好看"+"有人味/有态度"。0 分对照:好看但没人味(像品牌主图)。

### ID — Identity / 人设识别度(**涨粉地基维度**)
*这篇有没有透出一个**能被记住、想持续看**的人设/审美/态度?*

人设是涨粉的地基——账号靠它建立"我为什么要关注**这一个**而不是别的同类号"。视觉、口吻、价值观三者一致才立得住。

- **0** — 没有任何个人色彩,换个头像谁发都一样
- **3** — 有点风格,但不稳定 / 不鲜明,记不住
- **5** — 鲜明且一致的人设:统一的视觉调性(点开主页九宫格是一个人)、稳定的口吻、清晰的审美主张("反对叠戴堆料,极简银饰也能高级")。**看一篇就大概知道你是谁、主页还会有什么**

5/5 锚点:有人设标签能被一句话概括("那个只穿黑白、戴古着银饰的女生")。0 分对照:好看但没人设,刷完不记得是谁。

### FR — Follow Reason / 关注理由(**北极星维度**)
*这篇有没有给一个**关注下去的理由**——后面还有、值得蹲?*

关注 = 对未来的下注。一次性内容再好也只换来点赞,**连载感 / 系列 / 持续价值预期**才换关注。

- **0** — 单篇完结,没有"下次还有"的任何暗示
- **3** — 内容不错但孤立,关注与否随缘
- **5** — 明确的关注钩子:系列感("银饰避坑·第3期")、人设连载("记录我从0搭一个银饰衣橱")、明确预告("下期讲香水分装")、或主页强一致让人想"一次看个够"。**让人觉得"关注了不亏,后面还有好东西"**

5/5 锚点:结尾或选题自带"追更"理由。0 分对照:"今天就分享到这里"——看完即走。

### EM — Emotion / 情绪·态度·共鸣
*有没有戳中一种**情绪或态度**,让人觉得"对,我也这么想 / 我懂"?*

涨粉靠共鸣不靠干货。能让人产生情绪认同 / 价值观对齐的内容,才会让人想"关注这个跟我同频的人"。

- **0** — 平铺直叙报参数,没有任何情绪或观点
- **3** — 有一点态度,但不痛不痒
- **5** — 鲜明的情绪/态度/审美主张,能引发认同或讨论:"廉价感不是因为便宜,是因为没设计""我不在乎大牌,只在乎戴上像我自己"。**让人想点赞+评论+关注"这个跟我一国的人"**

### TR — Trust / 真实信任(不像广告)
*看起来像**真人真实分享**,还是像恰饭/通稿?*

口碑的命门。一旦像广告,涨粉和口碑同时崩。涨粉阶段尤其要"去商业感"。

- **0** — 通篇彩虹屁、无缺点、明显恰饭、像品牌话术
- **3** — 真实,但偶有"软广味"
- **5** — 真实使用痕迹 + 敢说缺点/真实取舍 + 个人化细节("这条我戴了两年发黑过,但我就喜欢这种旧感")。**没有挂车感,像朋友在掏心窝分享**

### SH — Shareability / 分享驱动(**口碑副信号维度**)
*有没有让人想**转发给朋友 / 安利出去**的理由?*

分享是口碑扩散的核心,也比收藏更能带来"朋友的朋友"这种高质量新粉。

- **0** — 没有任何"想转给谁"的点
- **3** — 有用但不至于专门转发
- **5** — 自带转发理由:替朋友说出了心声、太美想分享、观点犀利想@某人、实用到"必须转给闺蜜"。**让人愿意替你扩散**

### IH — Interaction Hook / 互动钩子(**口碑副信号维度**)
*有没有引导**评论/提问/站队**,且能引出**正向**讨论?*

小红书把评论互动当强权重,评论区温度还反哺曝光;评论的**正向率**是口碑的直接体温。

- **0** — 没有任何引导,结尾就没了
- **3** — 一句软引导("你们觉得呢")
- **5** — 具体可回答、能引出正向情绪的钩子:"报你的风格我帮你配银饰""你戴银饰被夸过最多的一句是什么""下期想看叠戴还是单戴,扣1扣2"。**让评论区有内容、有温度,而不是杠**

## 视频笔记附加维度(**仅当这篇是视频时加**)

小红书视频笔记叠加下面 2 个维度,图文笔记**跳过、不计入公式**。

### VH — Video Hook 3s(前 3 秒钩子)
*视频开头 3 秒能不能留住人、并立住人设?*
- **0** — 慢热铺垫、片头 logo、"大家好今天给大家分享"
- **3** — 直接进主体
- **5** — 第一帧就是态度/冲突/记忆点("我劝你别再买金饰了"直接怼脸 / 强人设出场),配封面级首帧

### VC — Completion / Pace(完播 / 节奏)
*时长和信息密度匹配吗,会不会拖?*
- **0** — 又长又水,大量废镜头
- **3** — 节奏一般
- **5** — 紧凑,15-40s 讲清一个态度/故事,每 2-3 秒一个信息或情绪变化

> 视频版公式:`composite = (CA + ID + FR + EM + TR + SH + IH + VH + VC) / 9 × 2.0`。
> 图文版用上面的 /7 版本。两种笔记在复盘时**分开记锚点**——它们的涨粉逻辑不同。

## Bucket 预测:小红书口径(比率桶,适用所有阶段)

> ⚠️ 小红书 0 粉新号笔记曝光典型几百到几千,用相对自己上一篇的**比率桶**,别套抖音绝对桶。
> bucket 仍以**曝光(view_count)**分档(和工具其余部分对齐、可比);**关注转化率 / 分享率 / 评论正向率单独作质量信号**看(见下一节),不混进 bucket。

### 第 1 篇:用小红书新号通用默认

| Bucket | 范围(实际曝光数)| 含义 | 先验概率 |
|---|---|---|---|
| 底部 | < 300 | 基本没进流量池 | 30% |
| 基础盘 | 300 - 2,000 | 小推荐,封面/标题及格 | 40% |
| 命中 | 2,000 - 10,000 | 进了二级流量池,有破圈苗头 | 20% |
| 小爆 | 10,000 - 100,000 | 共鸣/人设撑住的破圈 | 8% |
| 大爆 | > 100,000 | 进大流量池 | 2% |

### 第 2 篇起:比率桶(baseline = 上一篇曝光,或最近 3 篇中位数)

| Bucket | 倍数范围 | 含义 |
|---|---|---|
| 退步 | < 0.3 × baseline | 比上一篇明显差 |
| 持平 | 0.3 - 1 × baseline | 同档 |
| 命中 | 1 - 3 × baseline | 中度突破 |
| 小爆 | 3 - 10 × baseline | 显著破圈 |
| 大爆 | > 10 × baseline | 量级跃迁 |

> calibration_samples 少时,概率分布**应该更平**(如 30/30/20/15/5),诚实表达不确定。

## 质量带(**涨粉阶段看这个比看曝光重要**)

每次复盘除了 bucket,**必算三个口径**:

| 指标 | 算法 | 涨粉/口碑内容的健康参考带(先验,需用你自己数据重标) |
|---|---|---|
| **关注转化率(北极星)** | 新增关注 / 阅读 | < 0.5% 偏弱 · 0.5-1.5% 及格 · 1.5-4% 好 · >4% 很强 |
| **分享率** | 分享 / 曝光 | < 0.3% 偏弱 · 0.3-1% 及格 · 1-3% 好 · >3% 很强(强口碑) |
| **评论正向率** | 正向评论 / 总评论 | 看趋势:出现"关注你了""下期蹲""说出我心声"这类是强信号;杠/冷场是预警 |
| 收藏率(**降级·参考**) | 收藏 / 曝光 | 仍记录,但只当"有用度"参考,**不作目标**;若收藏高但关注转化低=被存了产品没被关注人 |

> ⚠️ 上面的带是先验占位,**不是你账号的真值**。跑完 5 篇用你自己的中位数重设(手动在 rubric_notes.md 记你账号的健康带)。
> **数据来源**:关注转化率的「单篇涨粉数」由 `xhs-explore` 适配器**自动抓取**(2026-06-27 已扩 `crawler.py` 接入创作者中心「数据看板→内容分析」的 `note/analyze/list` 接口,字段 `increase_fans_count`;report.md 直接给「涨粉 / 关注转化率 / 封面点击率」)。评论正向率由 cheat-retro 读 top 评论后判定。
> **核心判断**:一篇曝光一般但**关注转化率高**的笔记,比一篇曝光高但没人关注的笔记**更值得复制**——前者真的在长粉。

## T+N 复盘窗口:小红书要看两次

`RETRO_WINDOW_DAYS = 3`(默认,抓算法推荐期的结论)

**但小红书建议额外在 T+7d 再看一次**:搜索长尾会让一篇笔记在一周内持续被翻出来,**关注和分享也会继续缓慢累积**。只看 T+3d 会低估涨粉效果。

- T+3d:判断"算法推荐期"表现(曝光、即时关注转化)
- T+7d:判断"搜索长尾 + 口碑沉淀"(关注/分享通常此时才稳)

覆盖默认时记得写进 `rubric_notes.md`。

## Anchors(锚点样本)

打分新笔记时,找 2-4 个 composite 接近的旧笔记,列出它们的实际**关注转化率/分享率**对照。**锚点对比能抓出公式抓不到的错误**(比如"这两篇我都打 ID=5,但实际涨粉差 5 倍——人设差异在哪")。

锚点是你边发边攒的。≥5 篇校准前锚点不可用,这是正常的。Cold-start 期预测文件里标 `锚点对比:N/A(cold-start)`。

## Cold-start 战略(**必读**)

前 5 篇的"预测"不是预测,是**数据采集**。

**取样策略:主动选差异最大的笔记。** 别前 5 篇都发"同款",否则复盘时分不清是哪个维度在涨粉。刻意各发一篇侧重不同维度的:
- 1 篇 **ID 主导**(强人设/审美/视觉极致,内容轻)
- 1 篇 **FR 主导**(强连载/系列感,如"银饰避坑·第1期")
- 1 篇 **EM 主导**(强情绪/态度/观点,如"我为什么只戴银")
- 1 篇 **CA 主导**(封面/出片极致)
- 1 篇综合中等

复盘后你会看到 pattern(比如"我的态度款关注转化远超盘点款""我的封面分总高估")——这就是第一次 `/cheat-bump` 的证据。

**你不该做的**:cold-start 期看着 composite 8.4 就觉得这条会涨粉。精度 ±50%,决策没意义。已决定要发的发,犹豫的靠自己感觉。

| 校准样本 | 你能相信什么 |
|---|---|
| N=0-2 | 啥都别信,bucket 是占位 |
| N=3-5 | 信"哪个维度可能重要"的方向 |
| N=5-10 | 信 bucket 排序;不信中枢点估计 |
| N=10-20 | 中枢可信 ±30%,可作决策参考之一 |
| N≥20 | rubric 真正成为"作弊器" |

## 第 6 篇起的升级选项

跑完 5 篇校准后:
- **路径 A**:`/cheat-bump --propose "<权重调整>"` 自己拟合(涨粉号常把 ID/FR/EM 提权,CA 降权)
- **路径 B**:把**关注转化率**作为主回归目标重设权重
- **路径 C**:5 篇还看不出 pattern → 再跑 5 篇,等权 v0 在 10-15 样本前不丢人

## 这份 rubric 注定会打错的地方(已知 limitations)

1. **关注转化数据依赖创作者后台**:per-note 涨粉数从创作者中心「内容分析」接口抓(已接入,见数据来源);注意该接口 navigate-intercept 只覆盖最近约 10 篇,更老的笔记涨粉暂不可得(TODO 分页)
2. **人设是慢变量**:ID/FR 的效果要几篇连续看才显形,单篇打分容易低估"账号累积效应"——涨粉常是"第7篇突然起量"而非单篇决定
3. **共鸣有时效/圈层性**:EM 维度高度依赖你打中的人群,同一态度对不同圈层关注转化差很多,rubric 没捕捉圈层
4. **封面玄学**:CA 只能粗评,真实效果发出去才知道——多准备 2-3 版测
5. **不预测带货转化**:本 rubric 预测关注/口碑,**不是 GMV**;哪天要带货切回 `xhs-zhongcao-zero.md`
